## TA Application Management System

### Features
- Create, edit, and delete applications as Teacher
- Register, login, and logout
- Filter and sort applications by column
- Accept applications as TA
- And more

### Installing the application

    composer install
    npm i && npm run build
    php artisan migrate
    // (Optional) For the purpose of adding test users and applications
    php artisan db:seed

### Running the dev server

    composer run dev

### Technologies
- Laravel
- Bootstrap
- SQLite
- Tailwind CSS
- Axios
- Vite
- Composer
- NPM
